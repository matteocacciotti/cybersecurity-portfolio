# Network Components

Key hardware in any network influences how traffic is routed, monitored, or intercepted — all critical aspects for attackers and defenders.

## End Devices

- Examples: Computers, smartphones, IoT devices
- Generate or receive network traffic

## Intermediate Devices

- **Router** → Connects different networks and manages IP traffic
- **Switch** → Manages LAN traffic via MAC addresses
- **Firewall** → Enforces security policies
- **Access Point** → Extends wireless access
- **Call Manager** → Manages VoIP communication
- **Service Provider Switch** → Backbone switching for ISPs

## Network Media

- **Copper cables** (Ethernet)
- **Fiber-optic cables**
- **Wireless signals (Wi-Fi, Bluetooth)**
- **Cloud-based virtual links**
